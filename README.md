🍽️ Uber Eats Restaurant Analysis

📌 Project Overview

This project analyzes Uber Eats restaurant data using SQL and Streamlit.

🛠️ Tools Used

- Python
- MySQL
- Streamlit
- Pandas

📊 Features

- 15 business questions answered
- SQL-based analysis
- Data shown in table format

▶️ How to Run

pip install streamlit pandas mysql-connector-python
streamlit run app.py
